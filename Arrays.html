Unlike many other programming languages python doesn't have built-in support for arrays, python lists can be used as an alternative.

What is an array?

An array is a special variable which can hold more than one value at a time. These are seen in multiple programming languages. As mentioned prior python doesn't have built-in support for arrays, this is why lists are used. The example above shows how arrays are laid out in python.
