Functions

A function is a section of code that only executes when called. You can pass data, known as parameters, into a function. A function can return data as a result. You will see functions in all programming languages such as java and C++. These also play a part with determining what the program does and the overall output. IF you check any of our third-party websites or books we recommended, please look over functions multiple times as they will play a big part when you are programming.

Number of Arguments

By default, a function must be called with the right number of arguments. This meaning that if the function expects 2 arguments, you need to call the function with 2 arguments no more or less. If you try to call with anymore or less, you will recive an error.
