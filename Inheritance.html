Inheritance

Python, just like many other programming languages allows inheritance. This consists of a parent class and a child class. Child classes can inherit all methods and properties from a parent class.

By using inheritance, we can create a class that contains all the methods and attributes of another class. The class being inherited from, also known as the base class, is the parent class. The class that inherits from another class is referred to as a child class or derived class.

The __init__() function

We have so far produced a child class that receives its parent's properties and methods as inheritance. To the child class, we want to add the __init__() function (instead of the pass keyword). Every time the class is used to create a new object, the __init__() function is automatically called.

When you add the __init__() function, the child class will no longer inherit the parent's __init__() function. The child's __init__() function overrides the inheritance of the parent's __init__() function. To keep the inheritance of the parent's __init__() function, add a call to the parent's __init__() function, this is shown in the example above.
