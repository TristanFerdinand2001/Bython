Operators

Operators are used to complete operations with variables and values. In the following example, we use the + operator to a add together both integers. there are many types of operators such as Arithmetic operators, Comparison operators, Logical operators and many more. Operators, just like if and else statements are also a really important part of porgramming. We wont be looking over all of them but we will be looking over the one's you need to complete the next quiz.

Operators Groups:

Python divides the operators in the following groups:
  • Arithmetic operators
    •

    These operators are + (addition), - (subtraction), * (multiplication), / (division), and % (modulo).

  • Assignment operators
    •

    Assignment operators are used when it comes to assigning values or name to different types of data types.

  • Comparison operators
    •

    Comparison operators can compare numbers or strings and perform evaluations. Expressions that use comparison operators do not return a number value as do arithmetic expressions. Comparison expressions return either 1 , which represents true, or 0 , which represents false.

  • Logical operators
    •

    A logical operator is a symbol or word used to join two or more expressions so that the value of the compound expression produced depends only on that of the original expressions and on the meaning of the operator. Common logical operators include AND, OR, and NOT.
